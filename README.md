# OnTrack

A simple and efficient web application for tracking DB Railway time schedules and connections.

## Description

OnTrack is a lightweight tool designed to help users monitor and track Deutsche Bahn (DB) Railway time information. This application provides an intuitive interface for staying updated with your train schedules and journey times.

## Features

- **Real-time Tracking**: Monitor DB Railway schedules and timing information
- **User-friendly Interface**: Clean and intuitive design for easy navigation
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Lightweight**: Fast loading times with minimal resource usage

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Dahmeds/OnTrack.git
   ```

2. Navigate to the project directory:
   ```bash
   cd OnTrack
   ```

3. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local server for best results

## Usage

1. Open the application in your web browser
2. The interface will display DB Railway time tracking functionality
3. Use the provided controls to monitor your train schedules
